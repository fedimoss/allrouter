## 如何使用兑换券

### 1.创建兑换码

前提条件：已登录管理员账号。

（1）在控制台-"服务商"-"兑换码管理"中，点击"添加兑换码"开始创建兑换码；

![image](/docs_images/platform_1.png)

（2）设置兑换码，在弹出的侧边页面中依次设置兑换码的基本信息：

- 名称：自定义兑换码名称，如"老用户邀请福利"、"礼品卡50元券"等
- 过期时间：设置兑换码的有效期时间
- 金额：兑换码的额度
- 生成数量：兑换码的方法数量

设置完成基本信息后，点击"提交"、"确定"即完成兑换码创建。最后生成的兑换码*.txt文件下载后发送给对应用户即可使用。

![image](/docs_images/platform_2.png)

![image](/docs_images/platform_3.png)

![image](/docs_images/platform_4.png)

### 2.使用兑换码

前提条件：已登录用户账号，且已领取兑换码

（1）在控制台-"营销活动"-"兑换码"中，复制领取到的兑换码，将其输入到兑换码中，最后点击"立即兑换"完成兑换，接下来即可使用。

![image](/docs_images/platform_5.png)

![image](/docs_images/platform_6.png)

（2）兑换完成后，在"我的兑换记录"中可以查看兑换码的状态信息。

![image](/docs_images/platform_7.png)

## 如何设置模型定价

前提条件：已登录服务商账号

### 1.选择模型

（1）在控制台-"服务商"-"服务商管理"-"模型定价"中，在弹出的页面中点击"新增定价"。

![image](/docs_images/platform_8.png)

![image](/docs_images/platform_9_1.png)

或者点击"模型定价"，在弹出的页面中选择对应模型，然后点击"编辑"，直接修改对应加价比例、一级消费返佣比例（利润比例）、二级消费返佣比例（利润比例）。

![image](/docs_images/platform_9_2.png)

![image](/docs_images/platform_9_3.png)

### 2.设置模型定价

在弹出的页面中依次设置以下模型基本信息：

- 展示给服务商用户的模型名：填写模型名称
- 渠道价格：展示不同渠道下的模型定价，包括分组、计价类型、输入价格、补全价格、缓存读取价格、缓存创建价格、服务商折扣。
- 启用：打开启用
- 计价方式：
  - 按百分比加价：这里的加价基于服务商折扣价计算，不是基于主站原价。最终售价 = 主站原价 × 折扣 / 10 × (1 + 加价比例)，自定义设置加价比例。
  - 按固定差价：固定加价是在服务商折扣价基础上额外增加固定金额。最终售价 = 服务商折扣价 + 固定加价。自定义设置Token 模型加价倍率和按次模型加价金额。
- 一级消费返佣比例（利润比例）：自定义设置
- 二级消费返佣比例（利润比例）：自定义设置

设置完成后，点击"确定"即可。

![image](/docs_images/platform_10.png)

## 如何查看数据看板数据

前提条件：已登录管理员或普通用户账号

### 1.数据看板总览

（1）在控制台-"数据看板"中，可以查看以下数据信息：

- 24H消费后的余额
- 24H 使用大模型的请求次数统计、统计次数
- 24H消耗的额度和Token总数
- 性能指标的平均RPM和平均TPM数值
- 最近24H的模型资源消耗和请求趋势柱状图
- 平台模型调用的热度排行榜
- 模型数据分析数据，包括各模型调用额度占比、消耗趋势变化、调用次数分布、调用次数排行
- 常见模型的配置方法教程，包括OpenAI 配置教程、Claude Code 使用教程、Codex CLI使用教程

## 如何查看每次消费情况

前提条件：已登录用户账号

### 1.使用日志

在控制台-"操作日志"-"使用日志"中，可以查看并分析您的 API 调用详细数据和实时状态。

![image](/docs_images/platform_13.png)

### 2.消费详情

在控制台-"操作日志"-"使用日志"中，点击展开使用记录会显示每次请求的日志详情、计费过程和花费金额。

![image](/docs_images/platform_14.png)

前提条件：已登录用户账号

在控制台-"营销活动"-"邀请奖励"中，可以查看当前用户的待提取收益、累计总收益、成功邀请人数、邀请明细（被邀请人/注册时间/注册奖励/消费返利）。
